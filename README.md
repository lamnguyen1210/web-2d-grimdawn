# web-2d-grimdawn

A personal project for building a browser-based 2D action RPG engine inspired by games like Grim Dawn.

## Current Direction

- Camera-driven 2D world with click-to-move and direct movement support
- Action combat with skills, enemies, loot, and progression
- Data-driven systems so content can be authored without rewriting engine code
- Web-first architecture that can run locally in the browser and later be packaged for desktop

## Initial Priorities

1. Build the core gameplay loop in a vertical slice
2. Prove the engine architecture with maps, combat, AI, and loot
3. Keep tooling simple until the game feel is validated
